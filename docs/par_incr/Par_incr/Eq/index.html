Module Par_incr.Eq

val attach : fn:('a -> 'a -> bool) -> 'a t -> 'a t

Eq.attach ~fn:my_custom_eq_fn incr attaches my_custom_eq_fn as eq function to the incremental incr. This is useful in cases where you want somewhat different cutoff conditions. Take floating point related computations for example, you may choose to ignore difference in values within some delta. For such cases, you can use this.
